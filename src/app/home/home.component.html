Open Source Myanmar Software Tools

Open source projects for Myanmar specific software applications, libraries, utilities, language, script, Unicode and font tools by DagonMetric Myanmar Tools.

View Myanmar Tools on GitHub

Cross Platform Apps

Zawgyi Unicode Converter

Zawgyi Unicode Converter is a cross platform progressive web application (PWA) that automatically detects and converts Zawgyi to Unicode or Unicode to Zawgyi font text.

Unicode Code Point Lookup

Unicode Code Point Lookup is a Unicode converter (Unicode encode / decode) for converting UTF-32/UTF-16/UTF-8 code point sequences to text and vice versa.

Android Apps

Zawgyi Unicode Converter Android App

Zawgyi Unicode Converter Android App is a free & open source Zawgyi-One and standard Myanmar Unicode intelligent converter app / apk for Android.

Library Packages

DagonMetric/ng-translit

Transliteration service for Angular which can be used in swapping letters such as α → a, ၎ → ၎င်း or Zawgyi-One to standard Myanmar Unicode.

 myanmartools/zawgyi-unicode-translit-rules

Zawgyi Unicode transliterate / convert regular expression rules in JSON and JavaScript formats using DagonMetric/ng-translit.

 myanmartools/ng-zawgyi-detector

Zawgyi-One and standard Myanmar Unicode detector library for Angular applications.
