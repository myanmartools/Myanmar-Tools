Myanmar Tools

Open source projects for Myanmar specific software applications, libraries, utilities, language, script, Unicode and font tools by DagonMetric MyanmarTools. View Myanmar Tools projects on GitHub.

Web Applications

Zawgyi Unicode Converter

Free and open source Zawgyi-One and standard Myanmar Unicode online/offline converter.

Unicode Code Points Lookup

Unicode converter (encode/decode) for converting UTF-32/UTF-16/UTF-8 code point sequences to text and vice versa.

Android Applications

Zawgyi Unicode Converter Android App

Free and open source Zawgyi-One and standard Myanmar Unicode converter app for Android.

Library Packages

ng-translit

Powerful transliteration service for Angular applications which can be used in swapping letters such as α → a, ၎ → ၎င်း or Zawgyi-One to standard Myanmar Unicode.

zawgyi-unicode-translit-rules

Zawgyi Unicode transliterate/convert regular expression rules in JSON and JavaScript formats.

ng-zawgyi-detector

Zawgyi-One and standard Myanmar Unicode detector library for Angular applications.
